67.4. 例子

下面的命令集将创建名为test_table的表，它有两个列cola和colb，类型分别为int4和text的表，且表的OID为420， 然后向该表插入两行： 

create test_table 420 (cola = int4, colb = text)
open test_table
insert OID=421 ( 1 "value1" )
insert OID=422 ( 2 _null_ )
close test_table
