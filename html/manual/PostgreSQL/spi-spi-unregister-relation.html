SPI_unregister_relation

SPI_unregister_relation — 从注册表中删除一个临时命名的关系

大纲

int SPI_unregister_relation(const char * name)

描述

SPI_unregister_relation 从当前连接的注册表中删除一个临时的命名关系。

参数

const char * name

关系注册表项名称

返回值

如果命令执行成功，则将返回以下（非负）值：

SPI_OK_REL_UNREGISTER

如果关系名称已成功从注册表中删除

出错时，将返回以下负值之一：

SPI_ERROR_ARGUMENT

如果 name 是 NULL

SPI_ERROR_UNCONNECTED

如果从未连接的程序中调用

SPI_ERROR_REL_NOT_FOUND

如果在注册表中找不到当前连接的name
