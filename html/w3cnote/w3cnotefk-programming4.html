我的程序员之路 – 毕业找工作(大学结局)

分类 程序员人生

也不能说毕业找工作，现在在大学你最后一学期基本可以自由活动了，老师也是追求高就业率，恨不得你早点从学校滚蛋。

上一篇我讲到做毕业论文，其实论文我是在大四上学期的上半学期已经交稿了，也就做了一两个月，也建议大家不要把这个论文看的太重，能过就好，基本上你也做不出什么高深的东西。

然后这时候班级的同学有的忙着考研，有的忙着考公务员，有的忙着找工作去（我是其中之一）。

找工作的就到各个双选会，宣讲会投简历，去之前我们都会把自己的简历弄的漂漂亮亮的，什么字体啊，排版啊，颜色啊，现在想想挺可笑，好多同学（包括我）真材实料没有，简历到是弄的花枝招展。双选会上企业非常多，有私企、国企等。 印象比较深刻的是看到中国移动，感觉能进去就牛逼了，但是看到他们的两个招聘官就坐在椅子上嗑瓜子聊天，别人投了简历，头都不抬一下，貌似高人好几等的样子，当然他们桌子上也放了厚厚一叠简历，大家都想去这种大企业。

后来我从侧面了解到，这些企业基本是来走过场的，他们的招聘没有内部推荐你就别想了。

回到我这，我投了一些软件企业，如用友、金蝶、网龙等，结果页可想而知，完全没消息，基本上我投了十几家企业，没有一家找我面试（非计算机专业的也许就这个命）。

那会开始有点焦虑了，想到考了这么多证，学了这么久的编程（虽然还很菜），总不能跑去做其他门槛低的工作吧，那不是白学了。越想越郁闷，就想出去散散心，走着走着，突然看到学校门口附近（离大门200~300米）有一家在招聘 PHP 程序员，要求技术是 PHP 、MySQL、Ajax等，可是想到自己是学 Java 的，PHP 也不会，要求的技术也不懂，估计去面试人家也不会要我（悲观心态）。

过了一会，想想现在也找不到工作，如果真去做其他行业，不是被同班同学笑话了（全班级都知道我在学计算机），学了那么久还不是一样没用，这样也太没面子了，越想越不甘，不行，怎么着也得去试试。

后来自己也没多想拿了张简历，跑到那家招PHP的公司，进去之后，这公司也没前台，我就敲了敲玻璃门，一个戴眼镜的小哥跑出来问我什么事情，我说是来面试程序员的，然后他就招呼我道办公室坐下，问了我一些问题，大概就是什么专业，懂哪些技术，做过什么项目，我也如实回答说：我不会 PHP，我在学校自学计算机，会点 Java ，做过实验室的管理系统。之后他让我上机写下 JavaScript 代码，我虽然接触过，但反正也是不会做，当时看他也是挺无语的，之后他就让我回去等通知，我觉的是黄了……

但人生总是处处有惊喜，回到学校，那公司老板给我打电话了，说中午面试的小哥说我还可以培养，让我下周一去他们公司上班，工资一个月 1000（当时不给钱我也去，赚项目经验），过去后好好跟他们学习 PHP，当时我真的是震惊了，完全没想到，怎么可能，我接完电话后，发现手心都流汗了，估计是太激动了。之后，我就直接跑到图书馆借了本 PHP 入门的书猛看，各种安装环境，语法学习，离下周一还有三四天，恶补下……

第一天去公司的时候，面试的小哥给我安排了个位置，这小哥后来也是我主管，然后让我安装下PHP开发环境，那时我就用php集成环境phpnow 一键安装，点下一步、下一步……也就搞定了，之后主管甩了个地址给我，打开一看是织梦 CMS 官方网站（国内做网站的非常多），让我下载这个网站的代码，然后研究下源码跟模板开发，下载后，我从这个系统的 index.php（入口页面）开始看，然后看到各种php函数，也完全不懂，那时也刚接触PHP 3 天，又不好意思去问，就硬着头皮下载了 php 参考手册的 chm，然后边看系统逻辑结构，边看 chm 手册上的函数解释，学了一个月左右，我也慢慢帮公司做模板，开发插件了，就这样我就开始了 PHP 程序员的生涯……

总结

在你什么都不会的时候，主动去争取，即便最后什么都拿不到，做了也许没用，不做肯定没用，有时候机会就那么一次。

包括现在我们已经在公司工作的童鞋，公司的机会也非常多，与其等着领导来跟你说怎么做，倒不如你主动先做好这些事情，3~5年之后人与人之间的差距就在这 "主动" 二字。
