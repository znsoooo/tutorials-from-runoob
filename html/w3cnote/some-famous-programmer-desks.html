看国外“大神”程序员高大上的电脑桌

分类 科技资讯

估计，大部分阅读本文的程序员都是坐在敞厅的隔断里编程。这种工作环境是最节省空间的，但未必是最节省工作能量和注意力的。程序员不喜欢开放式办公室，因为开放式的办公室嘈杂，极易发生各种干扰事件，而程序员是最害怕被打搅的职业人群。但独立办公室并不是每个程序员都能有资格享受的，他们即使是坐在嘈杂的敞厅的隔断里编程，也是要按时完成工作。虽然吃不着猪肉，我们还是可以看看猪跑的。本文搜集了一些国外知名程序员的办公桌的样子，仅供大家围观。



1、Ilya Grigorik: Google 工程师。



2、Alex MacCaw: JavaScript 程序员，O'Reilly 作者，目前在 Stripe 公司工作。



3、Sam Soffes: iOS 和 Rails 软件工程师，Synthetic 前雇员，目前在创业。



4、Max Howell: Homebrew 的作者。



5、Scott Hanselman: 微软工程师。



6、Jeff Lindsay: 开源黑客，Hacker Dojo 创始人，目前在 Twilio 工作。



7、Dr Nic Williams: 每个 Ruby 社区的人都应该听过或看过他的作品，贡献了超过100个开源项目，目前在 Engine Yard 工作。



8、Phil Haack: 前微软 ASP.NET 程序经理，目前在 GitHub 工作。



9、Michael Bleigh: 开发了几个被广泛使用的 Ruby 开源软件库。



10、Chris Eppstein: Compass 框架作者和 Sass 贡献者。
